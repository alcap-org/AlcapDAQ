Introduction

hvEdit is a GUI written in C++ using the midas libraries and the Qt libraries from trolltech. For a description concerning this libraries see the 'table of contents'.

hvEdit uses the hotlink feature of midas. It actually only is linked to the online database (ODB), i.e. all value changes in hvEdit are transmitted to the ODB. From there the corresponding high voltage driver take care to set the desirable values.

hvEdit is based on the original hvEdit for Windows written in Visual C++ from Stefan Ritt. However it proved to be a handicap to be bound on Bill Gates Microsoft World. Therefore we decided to try to use Qt libraries to get linux GUI to fly.
