HOWTO take "WFD early start" run.

  1. Stop the DAQ:
    The best way to stop the run is to set "Restart" to "N" on MIDAS page and wait till the end of the run
  2. Unplug the "Kicker safe" cable.
  3. Stop MagicBox:
    Set "/Equipment/MB/Settings/Kicker/On" to "n" in ODB
  4. Set WFD start to proper value (Pulser05 in ODB):
    Set "/Equipment/MB/Settings/PulseReg/Pulser05/MB Clock Start" to 0
    Set "/Equipment/MB/Settings/PulseReg/Pulser05/MB Clock Start" to 200
  5. Start Kicker:
    Set "/Equipment/MB/Settings/Kicker/On" to "y" in ODB
  6. Plug the "Kicker safe" cable back.
  7. Check Kicker traces on the scope.
  8. Start the DAQ and take data.
  9. To return to the normal data taking run, repeat all these steps and "MB Clock start" to 234 instead of 200.

Two shell scripts
/home/mulan/DAQ/daq-scripts/set_WFDstart_early.sh and
/home/mulan/DAQ/daq-scripts/set_WFDstart_normal.sh
Will make changes in ODB for you! Just enter the directory /home/mulan/DAQ/daq-scripts/ and execute desired script.
